# 데이터 사이언스 프로세스

- 문제 정의하기
  - 해결하고자 하는 문제 정의
    - 목표 설정
    - 기간 설정
    - 평가 방법 설정
    - 필요한 데이터 설정  


- 데이터 모으기
  -  필요한 데이터 모을 수 있는 방법 찾기
    - 웹 크롤링
    - 자료 모으기
    - 파일 읽고 쓰기

- 데이터 다듬기
  - 데이터의 퀄리티를 높여 의미 있는 분석 가능하게 하기
    - 데이터 관찰하기
    - 데이터 오류 제거
    - 데이터 정리하기

- 데이터 분석하기
  - 준비된 데이터로부터 의미 찾기
    - 데이터 파악하기
    - 데이터 변형하기
    - 통계 분석
    - 인사이트 발견
    - 의미 도출

- 데이터 시각화 및 커뮤니케이션
  - 분석 결과를 다른 사람들에게 전달
    - 다양한 시각화
    - 커뮤니케이션
    - 리포트
